# Launchly
Seamlessly switch between Minecraft: Bedrock Edition versions with ease.

# Info
Platform: Android<br>
Supported Android Versions: Android 8 (Oreo) - Android 15 (Vanilla Ice Cream)<br>
Supported ABIs: armeabi-v7a, arm64-v8a, x86, x86-64
